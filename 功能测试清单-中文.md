# UU9体育直播站 — 功能测试清单（中文）

> 接手团队部署后，按此逐项勾选验证。
> 测试地址示例：`https://uu9-web.onrender.com`

## A. 基础 / 后端存活
- [ ] 打开 `/health` 返回 `{"ok":true}`
- [ ] 首页正常显示，导航 / 页脚 / Logo「UU9体育直播站」正确
- [ ] 切换 5 种语言（中/英/越/马/印尼）文字都变

## B. 首页内容
- [ ] 「正在直播」显示房间卡片
- [ ] 「赛程比分」热门/足球/篮球/电竞切换有数据
- [ ] 「积分榜」「热门资讯」有内容
- [ ] 顶部广告位 / 直播间下方广告位显示（或为空但不报错）

## C. 直播间 `live.html`
- [ ] 点房间进入，HLS 视频自动播放
- [ ] 网络中断后能自恢复（断流重连）
- [ ] 聊天可输入发送
- [ ] 😀 按钮弹出 100 表情面板，分 5 组
- [ ] 纯表情消息放大 + 飘屏动画
- [ ] 聊天顶部显示《文明观赛公约》
- [ ] 未登录满 5 分钟弹注册引导 + 倒计时横幅

## D. 注册 / 登录 `auth.html`
- [ ] 注册（邮箱+密码）成功
- [ ] 邮箱格式错 / 密码<6 位 有提示
- [ ] 重复邮箱提示已注册
- [ ] 登录成功，登录后直播间无 5 分钟限制
- [ ] 退出登录有效
- [ ] （若开启验证）收到/输入 6 位码才能登录

## E. 管理后台 `admin.html`
- [ ] 用 ADMIN_TOKEN 登录
- [ ] 新建房间（填标题/分类/.m3u8 或腾讯云流名/允许国家）
- [ ] 房间出现在首页
- [ ] 编辑 / 删除房间有效
- [ ] 「允许观看国家」设为某国后，其他国访问返回 451（需 CDN 注入国家头才能真测）
- [ ] 新建 / 编辑 / 删除广告，前台对应位置显示

## F. 开播台 `go-live.html`
- [ ] 「开启摄像头」请求权限并预览（须 HTTPS）
- [ ] 「共享屏幕」可切换
- [ ] 6 种布景切换
- [ ] 加文字 / Logo / 滚动条，可拖动，双击删除
- [ ] 「开播」计时 + 生成观众链接
- [ ] 「保存封面」导出图片

## G. 腾讯云对接（采购 CDN 后）
- [ ] `.env` 填 TENCENT_PLAY_DOMAIN / TENCENT_CDN_KEY / TENCENT_APP_NAME
- [ ] 后台房间只填流名，后端返回带签名 .m3u8
- [ ] 真实推流 → 直播间能播放
- [ ] 确认签名算法与控制台所选鉴权类型一致（本项目实现 TypeA）

## H. 上线前合规与硬限制（重要）
- [ ] 视频源具备合法版权
- [ ] 地理封锁在 CDN 层硬限制（非仅 Web 层）
- [ ] 播放令牌时效化（防直取 .m3u8）
- [ ] 邮件服务已接入
- [ ] 数据库换持久化（Postgres）
- [ ] 用户协议/隐私经律师审核
- [ ] 内容审核（鉴黄）已接入

> 注：本项目逻辑单元已通过单测，但未做真实服务器完整联调。任何一项失败请记录日志反馈。
